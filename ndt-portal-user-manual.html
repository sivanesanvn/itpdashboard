Overview

The NDT Portal is a web-based inspection management system for Cutech Singapore. It connects industrial clients, NDT technicians, coordinators, managers, and support contractors (scaffold, insulation, painting) on a single platform — from job request to final report delivery.

All users access the system through the same login page. The system automatically routes each user to their role-specific dashboard upon login.

How to Use — By Role

Client

  1. Log in → you are taken to My Requests.
  2. Click New Request to submit an inspection job.
  3. Complete Step 1: location, NDT method, priority, and plant details.
  4. Complete Step 2: additional job specifications and notes.
  5. Submit — your request appears in the list with status New.
  6. Monitor status updates from the same page as work progresses.

Manager

  1. Log in → you are taken to the Manager Dashboard.
  2. Go to Requests to view all incoming requests.
  3. Click Schedule on a New request → set date, assign technician, and add support jobs if required.
  4. Go to Technicians to check availability before scheduling.
  5. Go to Team to add new users, update roles, or manage profiles.
  6. Monitor job progress and report submissions from the dashboard.

Coordinator

  1. Log in → you are taken to the Coordinator Dashboard.
  2. View live status of all ongoing and pending jobs.
  3. Use this view to coordinate fieldwork and support activities.

Technician

  1. Log in → you are taken to My Jobs.
  2. View all jobs assigned to you with scheduled dates and locations.
  3. Update job status as work progresses on site.
  4. Upload the draft report once site work is complete.

Scaffold / Insulation / Painting

  1. Log in → you are taken to My Jobs.
  2. View only the support jobs assigned to your team.
  3. Update your support job status as work progresses.

Key Rules & Access Controls
  • Clients can only view and manage their own requests — other clients' data is never visible.
  • Technicians only see jobs that have been assigned to them by a Manager.
  • Contractors (scaffold, insulation, painting) only see requests where they have an active support job.
  • Only the Manager can schedule jobs, assign technicians, and manage user accounts.
  • All status changes are permanently recorded in the system audit trail.
  • Document uploads (reports, supporting files) are securely stored and linked to the request.
